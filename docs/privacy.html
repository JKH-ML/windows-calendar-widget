Privacy Policy

This privacy policy applies to the Windows Calendar Widget application developed and operated by JKH-ML.

Data Collection

The application accesses Google Calendar data such as event titles, start and end times, and event descriptions, only with the user's explicit consent.

Use of Data

The accessed data is used solely to display calendar events within the Windows Calendar Widget. The application does not use this data for advertising, analytics, or any other purpose.

Data Storage

The application does not store Google user data on any external servers. All data is processed locally on the user's device.

Data Sharing

The application does not share Google user data with any third parties.

User Control and Data Deletion

Users can revoke the application's access to their Google account at any time via the Google Account permissions page, which will immediately stop data access.
